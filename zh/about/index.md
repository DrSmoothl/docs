# 关于本项目

MaiBot（麦麦 / MaiSaka）是一个基于大语言模型的可交互智能体。她不只是那种"有求必应"的助手——她试着了解你，用接近真人的方式跟你相处。她不追求完美和高效，她追求的是亲切和真实。

- 💭 **没有人喜欢 GPT 的语言风格**：麦麦说话更自然，更像真人聊天——不是长篇大论，也不是 Markdown 分点罗列，而是有长有短的闲谈。
- 🎭 **不再是傻乎乎的一问一答**：她懂得看气氛说话，该开口时开口，该沉默时沉默。
- 🧠 **麦麦·成为人类**：在多人聊天里，麦麦会模仿其他人的说话风格，还会自己琢磨新词和圈子里的黑话，一直在进化。
- ❤️ **永远都在更加了解你**：基于心理学的人格理论，麦麦会不断积累对你的了解——你的信息、喜好、说话风格，她都记在心里。
- 🔌 **插件系统**：提供强大的 API 和事件系统，扩展空间无限。

## 设计理念

> **千石可乐说：**
> - 这个项目最初只是想给牛牛 bot 加点小功能，结果越写越多，最后干脆重写了。目标是在 QQ 群聊里创造一个"生命体"——不是一个功能齐全的机器人，而是一个能让你感觉到真实存在的家伙。
> - 功能设计围绕一个核心原则："最像，而不是最好"。
> - 如果一个人真的需要 AI 来陪伴，那未必是一个完美、什么都能解决的"helpful assistant"。也许更想要的，是一个会犯错、有自己想法和感受的"生命"。
