# 常见问题

这里收录使用 MaiBot 时经常遇到的问题。请先按问题所属分类查找；如果日志中已经出现明确错误代码，可以直接前往[错误排查](./error-troubleshooting.md)。

## 问题分类

**[基础使用](./basic-usage.md)** — 费用、支持平台、首次使用和获取帮助。

**[部署与启动](./deployment.md)** — 启动失败、配置文件、端口和 WebUI。

**[一键包](./one-key.md)** — 安装位置、用户数据、实例迁移和连接检查。

**[适配器连接](./adapters.md)** — NapCat、SnowLuma、聊天过滤和连接参数。

**[聊天与回复](./chat-and-reply.md)** — 发言频率、自我回复、连续消息和回复分割。

**[模型与 API](./models-and-api.md)** — API Key、模型标识符、视觉模型和 embedding。

**[记忆与学习](./memory-and-learning.md)** — 长期记忆、关系向量、错误记忆、表达与黑话。

**[插件问题](./plugins.md)** — 插件启用、命令冲突、下载失败和故障定位。

**[数据与迁移](./backup-and-migration.md)** — 备份范围、迁移步骤和恢复检查。

## 快速排查

遇到问题时，建议依次确认：

1. 记录 MaiBot、一键包和适配器的版本。
2. 查看问题发生时前后的完整日志，不要只截取最后一行。
3. 确认最近修改过的配置、模型和插件。
4. 按对应分类执行检查；有明确报错时查询[错误排查](./error-troubleshooting.md)。
5. 仍无法解决时，携带版本、日志、相关配置和复现步骤反馈。
